# Operations

**Created:** 2026-07-09  
**Last updated:** 2026-07-18

This folder holds my cross-system operational records, the material that spans more than one platform or infrastructure component:

- `Inventory/` - dated current-state and historical inventories
- `Maintenance/` - cross-system maintenance records
- `Diagnostics/` - raw diagnostic output such as SMART results

Detailed service changes and troubleshooting stay with the owning infrastructure component or platform. I keep the shared inventories and raw diagnostic evidence that support them here.
