# Operations

**Created:** 2026-07-09  
**Last updated:** 2026-07-09

Operations contains cross-system operational records:

- `Inventory/` — dated current-state and historical inventories
- `Maintenance/` — cross-system maintenance records when created
- `Diagnostics/` — raw operational results such as SMART output

Detailed service changes and troubleshooting remain with the owning infrastructure component or platform. Operations stores the shared inventory or raw diagnostic evidence that supports them.
