# Security

**Created:** 2026-07-09  
**Last updated:** 2026-07-21

This directory holds my cross-system security records:

- `Incidents/`: security and service-impacting incident reports.
- `Assessments/`: audits, reviews, & risk assessments.
- `Hardening/`: cross-system hardening standards, including the [Linux Host Baseline Standard](Hardening/Linux-Host-Baseline-Standard.md) I apply to every new Linux VM and LXC.


Splunk and Wazuh live under `Platforms/` because they are deployed services. Their incident, assessment, & cross-system hardening records link back here when needed.
