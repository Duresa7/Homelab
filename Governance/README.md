# Governance

**Created:** 2026-07-09  
**Last updated:** 2026-07-21

Governance contains one living documentation standard & three dated workspace change records. The standard defines where work belongs, what a change record must contain, how evidence maps to a step, & which dates stay in a filename.

- [Documentation standard](Documentation-Standard.md): three documentation tiers plus the evidence, incident, placeholder, filename, & metadata rules I use
- [Change records](Change%20Records/): dated records of workspace-level reorganizations
